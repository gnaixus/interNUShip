## About
InterNUShip is a smart internship-matching platform that uses a dating-app-inspired algorithm to connect students with employers based on skills, preferences, and compatibility, ensuring the perfect internship fit for both sides. The web-based application is designed to centralise relevant internship opportunities into a single platform, streamlining the application process by tracking application deadlines, facilitating document submissions, saving internship listings and interacting with other users in a community network.

## Getting Started 
```
# Clone your repository
git clone https://github.com/gnaixus/interNUShip.git
cd interNUShip

# Start frontend 
cd frontend
npm install
npm start

# Start backend 
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
